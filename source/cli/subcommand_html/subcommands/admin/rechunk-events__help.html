usage:
    ctc rechunk-events [CONTRACT] [EVENT] --chunk-bytes CHUNK_BYTES [options]

description:
    rechunk events by specific chunk size

arguments:
    CONTRACT                     address of contract emitting the event
    EVENT                        event hash
    --network NAME_OR_ID         network to rechunk events of
    --all                        whether to rechunk all events (can take a long time)
    --start-block START_BLOCK    start of block range to rechunk
    --end-block END_BLOCK        end of block range to rechunk
    --chunk-bytes CHUNK_BYTES    target number of bytes per chunk
    --dry                        perform a dry run where no changes are made
    --verbose                    increase verbosity
