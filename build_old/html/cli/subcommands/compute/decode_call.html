usage:
    ctc decode call ARGS [options]

description:
    decode EVM call data
    
    syntax is one of
        1. ctc decode call CALL_DATA
        2. ctc decode call CONTRACT_ADDRESS CALL_DATA
        3. ctc decode call TX_HASH --tx
